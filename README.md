# Swiggy Analytics Dashboard (Excel)

## 📊 Project Overview
This project focuses on analyzing Swiggy food delivery data to understand sales performance, order trends, and restaurant contributions. The dashboard provides an interactive view of business metrics to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales and revenue trends  
- Track order volume and delivery patterns  
- Identify top-performing cities and restaurants  
- Evaluate operational efficiency  

---

## 📁 Dataset
- Swiggy dataset (Excel)
- Includes:
  - Order details  
  - Restaurant data  
  - City-wise performance  
  - Revenue and sales metrics  

---

## 🛠 Tools & Technologies
- Excel / Power BI  
- Data Cleaning & Transformation  
- Data Visualization  

---

## 📈 Dashboard Features
- Sales and revenue tracking  
- Order count analysis  
- City-wise performance breakdown  
- Restaurant-wise contribution  
- Time-based trend analysis  
- Interactive filters and slicers  

---

## 🔍 Key Insights
- Identified top-performing cities contributing highest revenue  
- Analyzed restaurant-wise sales to find key contributors  
- Observed peak order timings and demand trends  
- Highlighted operational inefficiencies in order processing  

---

## 📂 Repository Contents
- `swiggy-dashboard.xlsx` → Excel dashboard file  
- `raw-data.xlsx` → Dataset used  

---

## 🚀 How to Use
1. Download the Excel file  
2. Open in Microsoft Excel  
3. Use filters/slicers to explore insights  

---

## 📌 Conclusion
This project demonstrates how Excel/Power BI can be used to analyze food delivery data and extract meaningful business insights for better decision-making.
